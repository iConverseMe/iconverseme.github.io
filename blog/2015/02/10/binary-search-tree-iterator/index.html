[LeetCode] Binary Search Tree Iterator

Problem

Implement an iterator over a binary search tree(BST). Your iterator will be initialized with the root node of a BST. Calling next() will return the next smallest number in the BST. here

Solution

  1. use a list to save inorder nodes of the tree, and a index point to get the next element. Time complexity for next() & hasNext() is O(1), space complexity is O(n), n is count of tree nodes.
  1. use a stack, the top of the stack is the next smallest, and each time pop up the top element, push the node’s right child’s left -> left –> leftest node to stack. Time complexity is O(1), space complexity is O(lgn). lgn is the height of the tree.

The Solution is the same as implementing inorder iteratively. Another way to inplement this.
